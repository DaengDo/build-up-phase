### DirectX12 대응
- 디스크립터 				<-> 디스크립터(뷰)
- 디스크립터 풀			<-> 디스크립터 힙
- 디스크립터 셋			<-> X
- 디스크립터 셋 레이아웃	<-> 루트파라미터, 디스크립터 테이블
- ?다이나믹 오프셋?		<-> 루트파라미터, 인라인 디스크립터
- ?푸시 상수?			<-> 루트파라미터, 인라인 상수
- 파이프라인 레이아웃		<-> 루트시그니쳐
